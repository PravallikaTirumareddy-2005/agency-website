# Digital Agency Website

A fully responsive digital agency website built using HTML, CSS, and JavaScript.

##  Features

- Responsive modern UI
- Mobile navigation menu
- Scroll reveal animations
- Interactive FAQ section
- Sticky navbar with scroll effect
- Real-time contact form integration using EmailJS

##  Tech Stack

- HTML5
- CSS3
- JavaScript
- EmailJS

##  Contact Form

The contact form is integrated with EmailJS to send real-time email notifications directly to the admin without requiring a backend.
